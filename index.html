Q1. " function counter(){

        function IncreaseCounter() {
            return counter += 1;
        };

    }
    
    var counter = counter();
    alert(counter());
    alert(counter());
    alert(counter());
    alert(counter());"

o/p: 1 2 3 4 

REASON: In the above code we have declared two functions: counter and Increase counter , where counter is the outer function and Increasecounter is the inner function ,
        in the counter function , we have declared a variable called counter with value =0, and in the increasecounter function we have incremented the variable that has been previously declared in the 
        counter function. every time the counter function is called, it returns a function with increment of the variable counter, hence the output will be incremented by 1.
